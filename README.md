# Creative Bus — лендинг рекламы на транспорте

Сверстан по макету Figma: [Mary → node 6441-2010](https://www.figma.com/design/o1syNp93H3v2dyA3JHp4em/Mary?node-id=6441-2010).

## Запуск

```bash
cd ~/projects/agvtrans-landing
python3 -m http.server 8013
# открыть http://localhost:8013
```

Без билда: чистый HTML/CSS/JS, шрифт Inter с Google Fonts.

## Структура

```
index.html   — вся страница (hero, услуги, кейсы, о нас, маршруты, CTA, FAQ, форма, футер)
styles.css   — стили, mobile-first брейкпоинты: 640 / 1024 / 1280
script.js    — аккордеоны, мобильное меню, маска телефона, валидация формы
assets/      — картинки, выгруженные из Figma + pricing.pdf (заглушка)
```

## Что работает

- Аккордеон услуг (одна открытая группа, по макету раскрыта «Рекламная наклейка»)
- FAQ-аккордеон
- Мобильное меню (бургер)
- Маска телефона `+7 (9XX) XXX XX XX`, валидация (имя ≥2 симв., телефон полный, согласие)
- Отправка формы — заглушка: `console.log` + success-экран. TODO в script.js — подключить email/Telegram, когда будут доступы
- Кнопка «наверх» в футере
